The SkillGauge System is a Java-based online quiz application that allows users to test their knowledge in different subjects. The system starts with a login and registration feature, where users can create an account or log in using their username and password. After logging in, the user can select a quiz category such as Java, DBMS, C Programming, or General Knowledge. The user also selects the difficulty level: Easy, Medium, or Hard. The system then displays a set of multiple-choice questions based on the selected category and difficulty level. Each question has four options, and the user must select the correct answer. The quiz also includes a 10-second timer for each question, which makes the quiz more interactive and challenging. At the end of the quiz, the system calculates the total score, number of correct answers, wrong answers, and percentage. Based on the percentage, the system assigns a grade (A, B, C, or D) and provides performance feedback to the user. Overall, the SkillGauge System helps users evaluate their knowledge, practice quiz questions, and improve their skills in different subjects.
